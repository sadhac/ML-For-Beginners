# Wsparcie
## Jak zgłaszać problemy i uzyskać pomoc  

Przed zgłoszeniem problemu sprawdź nasz [Przewodnik rozwiązywania problemów](TROUBLESHOOTING.md), gdzie znajdziesz rozwiązania typowych problemów związanych z instalacją, konfiguracją i uruchamianiem lekcji.

Ten projekt wykorzystuje GitHub Issues do śledzenia błędów i próśb o nowe funkcje. Przed zgłoszeniem nowego problemu przeszukaj istniejące zgłoszenia, aby uniknąć duplikatów. W przypadku nowych zgłoszeń, zgłoś swój błąd lub prośbę o funkcję jako nowe Issue.

Aby uzyskać pomoc i odpowiedzi na pytania dotyczące korzystania z tego projektu, możesz również:
- Sprawdzić [Przewodnik rozwiązywania problemów](TROUBLESHOOTING.md)
- Odwiedzić nasz [kanał Discord Discussions #ml-for-beginners](https://aka.ms/foundry/discord)
- Zgłosić problem
